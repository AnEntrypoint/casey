casey src/case-tools.js: ownsCase(externalId, author) is the single shared helper for case-ownership checks (external_id contains the author as a colon-separated part), used by both case_get's owns check and mineRows' mine filter -- previously two independent reimplementations that could diverge and reopen a PII leak if only one were fixed. Any new ownership check in case-tools.js should call this helper. Fixed 2026-07-10, commit 4cd7784.
